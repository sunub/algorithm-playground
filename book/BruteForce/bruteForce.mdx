# Brute Force 전체 탐색을 배우는 의미

선형 탐색법(linear search method)은 요소를 하나하나 순서대로 조사하는 방식입니다. 그리고 이러한 선형 탐색법에서 주로 이용되는 것이 `플래그(flag)`가 있습니다. 조건을 달성했더라도 탐색을 이어가는 것은 크게 의미가 없기 때문에 특정 조건을 달성하였을 경우 플래그 변수를 변화를 주어 조건을 만족시키는 방법입니다.

# 부분합

부분합을 탐색하는 방법으로 2진법 표현인 비트 연산을 사용하는 방법이 유용합니다.

`a = [1, 2, 3, 4, 5, 11]` 의 집합 내부에 있는 숫자를 이용하여 만들 수 있는 수의 집합의 조합들의 개수는 2^ N 개가 나옵니다. 2^N개에는 공집합이 포함한 총 숫자입니다.

# 부분합에 제약 조건이 붙을 경우

부분합의 또다른 방법은 트리의 형태로 구성하는 것입니다. 집합 내부의 모든 요소들은 선택을 할지 선택을 하지 않을 지에 대한 2가지의 경우로 나뉩니다. 이러한 경우에서 추가적인 제약조건으로 각각의 집합의 합이 W가 초과 되지 않을 경우에 대한 조건이 붙는다면 어떻게 해야 할까요?

```javascript
function func(i, w, a) {
    if (i === 0) {
        if (w === 0) return true;
        return false;
    }

    if (func(i - 1, w, a)) return true;

    if (func(i - 1, w - a[i - 1], a)) return true;

    return false;
}
```

# 최대 공약수

재귀 함수의 예시는 유클리드 호제법이 있다.
유클리드 호제법은 두 정수 m,n의 최대 공약수인 GCD(m, n)이 있을 경우 m을 n으로 나눈 나머지인 r은 GCD(m, n) = GCD(n, r)을 만족한다는 공식이다. 이러한 성질을 이용하여 최대 공약수를 구해보면

1. m을 n으로 나눈 나머지를 r이라고 한다.
2. r = 0이며 이 시점에서 n이 구하는 최대 공약수이므로 n을 출력하고 출력한다.
3. r != 0이면 n -> m, r -> n으로 지정하고 1로 되돌아 간다.

```javascript
function GCD(m, n) {
    if (n === 0) return m;

    return GCD(m, m % n);
}
```
